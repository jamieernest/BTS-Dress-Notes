    How It Works

    • Start typing - your timecode and LX cue freeze
    • Send note - it gets timestamped with frozen timecode and LX cue
    • Add tags to categorize your notes
    • Anyone can edit tags on any note
    • Add comments to notes for discussions
    • LX cues automatically update via OSC from EOS
    • Filter notes by tags to find what you need
    • Export all notes as JSON or CSV with timestamps
    • Toggle between Real Time and MIDI Timecode (Global)
    • Auto-resume after 15 seconds if note is empty and not focused
    • Use the "View Exported Notes" link to review past sessions
